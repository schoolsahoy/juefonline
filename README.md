# juefonline.org — JUEF Website

**Jain Universal Education Foundation**
Create Capable Humans for the AI Age through 2,500 years of Shraman Wisdom.

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Zero dependencies, zero build step
- Mobile-first responsive design

## Quick Start
Open `index.html` in any browser. No build tools required.

## Structure
```
├── index.html          Homepage
├── jeep.html           JEEP Landing Page
├── pages/              Content pages
├── css/style.css       Design system
├── css/responsive.css  Mobile breakpoints
├── js/main.js          Interactions
├── images/             Site images
└── assets/             Downloadable files
```

## Deployment
Configured for GitHub Pages with custom domain `juefonline.org`.
